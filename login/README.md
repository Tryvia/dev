# Login Page - Grupo Empresarial

Uma página de login moderna com tema de universo.

## Estrutura de Arquivos

```
standalone/
├── index.html      # Estrutura HTML
├── styles.css      # Estilos CSS
├── script.js       # JavaScript
├── images/         # Pasta com as logos
│   ├── logo-group.png
│   ├── logo-compass.png
│   ├── logo-target.png
│   ├── logo-globe.png
│   └── logo-clock.png
└── README.md       # Este arquivo
```

## Como Usar

1. Baixe todos os arquivos mantendo a estrutura de pastas
2. Abra o `index.html` no navegador

## Personalização

- **Cores**: Edite as variáveis CSS no início do arquivo `styles.css`
- **Logos**: Substitua as imagens na pasta `images/`
- **Textos**: Edite diretamente no `index.html`
- **Funcionalidade**: Modifique `script.js` para integrar com seu backend

## Requisitos

- Navegador moderno (Chrome, Firefox, Safari, Edge)
- Nenhuma dependência externa necessária
